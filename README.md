# Agro Control - Sistema de Gestão de Safras

## Descrição do projeto

O sistema Agro Control foi desenvolvido em Python com objetivo de gerenciar informações relacionadas à produção agrícola. Esse sistema permite o cadastro, edição, remoção e consulta de dados das safras. Além disso, também é possível gerar relatórios com indicadores de desempenho.

## Problema abordado

O setor do agronegócio, apesar de altamente relevante para a economia do país, ainda enfrenta diversas dificuldades relacionadas à gestão e ao controle de dados.

Os materiais de referência, especialmente o que diz sobre o contexto da produção de cana-de-açúcar, explicitam como a falta de controle e planejamento podem resultar em grandes perdas nas colheitas, afetando diretamente a eficiência do processo produtivo.

## Solução proposta

Diante dessa problemática e baseado nos conceitos de agrotech, o sistema Agro Control foi desenvolvido como uma solução tecnológica simplificada.

Em geral, esse sistema permite:

- Registrar informações das safras
- Organizar esses dados
- Calcular automaticamente indicadores (são expostos no relatório)

Por permitir o armazenamento organizado das informações e calcular indicadores, o Agro Control facilita na tomada de decisões.

## Funcionalidades do sistema

- Cadastro de safras
- Listagem de safras
- Edição de registros
- Remoção de registros
    - Salvamento e carregamento dos dados em JSON
- Cálculo de relatório (produção total, custo total e produtividade média)
    - Geração de relatório também em arquivo de texto
